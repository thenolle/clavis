# Code of Conduct

## Our Pledge

We aim to foster a welcoming, respectful, and productive environment for contributors and users of Clavis. This applies to all project spaces, including issues, pull requests, discussions, and any related communication channels.

## Expected Behavior

Participants are expected to:

- Be respectful and considerate in communication
- Focus on constructive feedback and technical merit
- Respect differing opinions and approaches
- Accept responsibility for mistakes and learn from them
- Use clear and professional language

## Unacceptable Behavior

The following behaviors are not acceptable:

- Harassment, insults, or personal attacks
- Discriminatory remarks of any kind
- Trolling or disruptive behavior
- Publishing private information without permission
- Persistent off-topic or spam content

## Scope

This Code of Conduct applies within all Clavis-related repositories, issues, pull requests, and associated communication spaces.

## Enforcement

Violations may result in:

- Warning
- Temporary restriction of participation
- Permanent removal from project collaboration

Enforcement decisions are made at the discretion of the project maintainers and will be applied fairly and consistently.

## Reporting

If you experience or witness unacceptable behavior, report it privately to the maintainers via GitHub security/contact channels. Reports will be handled confidentially.

## Acknowledgment

By participating in this project, you agree to abide by this Code of Conduct.
